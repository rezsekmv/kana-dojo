## Common Mistakes to Avoid

### ‚ùå App-Hopping
**Problem:** Constantly switching apps without giving any a fair chance.
**Solution:** Commit to one primary app (like KanaDojo) for at least 3 months.

### ‚ùå Ignoring Writing Systems
**Problem:** Using only romaji or skipping kanji.
**Solution:** Master Hiragana, Katakana, then Kanji systematically.

### ‚ùå Passive Consumption
**Problem:** Only reading/listening without active practice.
**Solution:** Use active recall (typing, speaking, writing).

### ‚ùå No Speaking Practice
**Problem:** Only using apps, never speaking.
**Solution:** Add language exchange after building foundation.

### ‚ùå Unrealistic Expectations
**Problem:** Expecting fluency in 3 months.
**Solution:** Set realistic goals, celebrate small wins.
